# ✅ To-Do List

A simple and intuitive task management app built with pure JavaScript. Add, filter, and delete tasks quickly, and mark them as completed.

## ✨ Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🔍 Filter tasks (all / active / completed)
- 💾 Data stored in `localStorage`
- 📱 Responsive design

## 🛠️ Built With
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
